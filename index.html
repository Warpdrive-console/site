What is Warpdrive

Built Different.
On Purpose.

🌐
Open Source Runtime

Web games via Chromium with WebGPU. Native ARM binaries later down the line for full-performance titles, and the freedom to choose your game engine. From unity, or GMS2, to Scratch, GBStudio, and GDevelop, with dozens of tools in-between.

🎮
Real Culture

The console Developers grew up with Newgrounds, Itch.io, and Gamejolt. PS3, Xbox360, and Wii modding culture. Open firmware. Moddable frontend, and a homebrew launcher out of the box. Warpdrive wouldn't exist without console-modding.

💿
Physical Media

Full-size DVD in purple collector cases. Warpdrive has a disc drive for physical games. It exists for collectors, or those without Wi-Fi. And hey, you can install the discs to storage, too.
